## ภาพรวมโครงการ
**Tastes Through Network (TTN)** คือระบบส่งรสชาติผ่านเครือข่าย ทำให้ข้าวเปล่ากินแล้วมีรสชาติเหมือนแกงเขียวหวานได้ โดยส่งข้อมูลรส กลิ่น และภาพไปกระตุ้นอุปกรณ์ที่ผู้ใช้สวมใส่

│ - แสดงภาพผ่าน AR │

## สรุปสั้นๆ แต่ละชั้น

| ชั้น | ชื่อ | หน้าที่หลัก |
|------|------|------------|
| 5 | Application | ให้คนเลือกเมนูที่อยากกิน |
| 4 | Transport | ส่งข้อมูลรสชาติแบบเร็วๆ ผ่าน UDP |
| 3 | Network | หาเส้นทางส่ง data ให้ถึงไวสุด |
| 2 | Data Link | บอกว่าอุปกรณ์ไหน ใครส่ง |
| 1 | Physical | อุปกรณ์จริงที่กระตุ้นลิ้น/จมูก/ตา |

## ตัวอย่างการทำงาน
1. คนกินเปิดแอพเลือก **"แกงเขียวหวาน"** (ชั้น 5)
2. ข้อมูลรสชาติถูกส่งแบบ UDP (ชั้น 4)
3. ระบบหาเส้นทางส่งที่ latency ต่ำที่สุด (ชั้น 3)
4. เพิ่มที่อยู่ผู้ส่ง-ผู้รับ (ชั้น 2)
5. อุปกรณ์ NPI กระตุ้นลิ้นให้เคี้ยวข้าวแล้วรู้สึกเหมือนแกงเขียวหวาน (ชั้น 1)
